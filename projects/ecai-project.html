ECAI
Exam Correction AI

An innovative AI-powered academic grading companion. Built specifically for teachers, ECAI automates exam transcribing and grading from paper sheets. Using a vision-capable LLM and advanced image pre-processing, it extracts handwritten or printed student answers, evaluates them against a PDF marking scheme, and generates detailed breakdowns and grades in seconds.

System Capabilities

Key Features

Three-Step Scan and Grading Flow
Captures student answer pages, verifies readability, uploads questions PDF, and associates them with a classroom/student record.
Real-Time Image Quality Assessment
Utilizes OpenCV on the backend (Laplacian variance and brightness analysis) to instantly flag blurry or poorly lit images before processing.
Advanced AI Grading Pipeline
Converts question key PDFs into images using PyMuPDF (fitz) and packages them with base64 student answer images to feed Mistral's vision model.
Structured JSON Evaluation Schema
Generates precise question-by-question grade breakdowns, rounding points to the nearest 0.25 and summarizing comprehensive constructive feedback.
Visitor / Demo Mode
Enables users to instantly test the AI's grading power. Upload a question PDF and answer photos to obtain immediate correction reports without registering.
Roster & Classroom Management
Full administration suite including subjects CRUD, classroom mapping, roster student registration (with CSV/bulk upload), and progress metrics.

Architecture

Under the Hood

ECAI is built on a decoupled Client-Server architecture. The Flutter client utilizes ChangeNotifier-based State Controllers linked via Provider to maintain clean separate UI states.

The Python server processes heavy LLM grading calls asynchronously as FastAPI BackgroundTasks, storing file references securely on local disk and updating Postgres asynchronously. Mistral Vision integration utilizes JSON mode enforcing predictable, clean schema replies.

How It Works

The AI Grading Pipeline

📸
1. Preprocessing

OpenCV checks Laplacian variance for blur and measures pixel brightness. Flurry or dark sheets are rejected immediately to guarantee AI accuracy.

📄
2. PDF Processing

PyMuPDF parses the master exam PDF key, rendering individual pages to high-density PNG arrays to supply legible visual queries.

🤖
3. LLM Vision Execution

Base64 encoded student copies and rendered PDF key pages are bundled together into the Mistral Vision model via token-aware HTTPX calls.

🎯
4. Structured Response

The model extracts coordinates, transcribes answers, and applies grading guidelines to produce a strictly structured JSON card, updating student database records.
