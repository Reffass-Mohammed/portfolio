What It Does

Key Features

Authentication
Email/password login & registration + Google Sign-In via Firebase Auth
Product Browsing
Home feed with banner slider, categories (Clothes, Bags, Shoes, Groceries…) and product detail pages with size/quantity selection
Cart & Favorites
Add items to cart with quantity control, save favorites and remove them with a swipe
Checkout Flow
Full checkout: delivery address selection (city-country search), payment card management, order summary with total
Push Notifications
Firebase Cloud Messaging integration with runtime permission handling on Android 13+
Native Splash Screen
Custom branded splash using flutter_native_splash with a purple theme
Stack

Technologies Used

🐦 Flutter 3.x 🎯 Dart 🔥 Firebase Auth 📦 Cloud Firestore 📬 Firebase Messaging 🔍 Google Sign-In 🧱 BLoC (flutter_bloc) 🖼️ Cached Network Image 🎠 Carousel Slider 🧭 Bottom Nav Bar
Architecture

Design Pattern

The app follows the BLoC (Business Logic Component) pattern for state management, cleanly separating UI from business logic. Firebase Firestore provides real-time data sync, while Firebase Auth handles secure identity management.
